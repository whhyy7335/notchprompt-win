```md
Hi Saif,

I really liked Notchprompt and built an unofficial Windows remake inspired by it.

The Windows version is open source here:

https://github.com/whhyy7335/notchprompt-win

It currently supports:

- Windows system tray workflow
- top-centered always-on-top teleprompter overlay
- start/pause/reset/jump back
- speed, countdown, font size, width, and height controls
- text import/export
- global shortcuts
- best-effort screen-capture exclusion on Windows

I credited the original Notchprompt project clearly in the README and included the original MIT license notice in the third-party notices file.

Thanks for the original idea and macOS implementation. If you prefer any wording around attribution, naming, or linking, I’m happy to adjust it.
```
